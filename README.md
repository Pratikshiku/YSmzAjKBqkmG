1. 登录
2. 注册
3. 首页
4. 优秀项目
5. 项目信息
6. 评审方案
7. 大创资讯
8. 个人中心

**指导老师功能：**
1. 登录

5. 项目类型管理
6. 项目信息管理

8. 项目中检管理
9. 项目结项管理
10. 项目评审管理
11. 专家评审管理
12. 评审方案管理

**院系管理员功能：**
1. 登录
2. 注册
3. 个人中心9修改密码、个人信息）

6. 项目信息管理

9. 项目结项管理

11. 专家评审管理
12. 评审方案管理




**管理员功能：**
